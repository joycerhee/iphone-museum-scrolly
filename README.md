# iPhone Evolution Museum

A scrollytelling exhibit built with Next.js, framer-motion, and Tailwind CSS. 
Each iPhone generation from 2007 to 2025 animates into view as you scroll, 
creating an immersive museum experience.

## Design Decisions

- **Design Style:** Minimalism — Apple-inspired white space, clean sans-serif typography, strict grid
- **Brand Archetype:** The Creator — visionary, innovative tone throughout
- **Cialdini Principle:** Social Proof — adoption stats and cultural milestones on every entry
